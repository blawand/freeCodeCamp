    • Setting up HTML

    HTML (HyperText Markup Language) is the skeleton of any webpage. Learn how to set up a basic HTML document with essential tags like <html>, <head>, <title>, and <body>

    Every HTML page starts with the tag, with the section containing metadata and links to CSS files, and the section containing all content visible to users. Start by creating a simple HTML structure:

    <!-- This is the root of the HTML document -->
<html>
  <!-- Head section contains metadata and title -->
  <head>
    <title>Welcome to My Site</title>
  </head>
  <!-- Body section contains the visible page content -->
  <body>
    <h1>Hello World!</h1> <!-- This is a main header -->
  </body>
</html>
    Headers and Paragraphs

    Explore how to use header tags (<h1> through <h6>) to create engaging headings and use paragraph tags (<p>) to organize your text content effectively.

    Organize your content with headers and paragraphs. Use <h1> to <h6> for headers: 

    <h1>Main Title</h1> 
  <p>This is a paragraph explaining the main topic.</p>
    CSS Syntax

    Cascading Style Sheets (CSS) control how your web pages look. Understand the basics of CSS syntax, including selectors, properties, and values to style your HTML elements.

    Ensure your styles are applied by linking your CSS file to your HTML document using the element inside the <head>

    <head>
  <link rel="stylesheet" href="styles.css">
</head>

    Learn to apply basic styles like color and font size.

    body {
  color: #333;
  font-family: Arial, sans-serif;
}

    Learn how to control layout with CSS properties like margin, padding, and border. These properties help you position and space elements precisely on your page.

    h1 {
  background-color: #4CAF50;
  color: white;
  padding: 10px;
  margin: 20px;
  border: 2px solid black;
}
    Simple styling

    Get creative with CSS by learning to apply simple styling techniques. Change colors, fonts, and spacing to enhance the visual appeal of your web pages.

    Enhance your web pages by styling elements. Change background colors and text styles easily:

    h1 {
  background-color: #4CAF50;
  color: white;
}
    Interactive Elements

    Enhance user interaction using HTML forms and CSS animations. Learn how to create forms for user input and animate elements to make your web pages more dynamic.

    Make your web pages interactive. Add forms for users to submit data.

    <form>
  <input type="text" placeholder="Enter your name">
  <button type="submit">Submit</button>
</form>
    Media Queries

    Media queries make your website accessible on devices of all sizes. Here's a simple example that adjusts text size based on the screen width:

    @media (max-width: 600px) {
  body {
    font-size: 14px; /* Smaller text for smaller screens */
  }
}

    Here are 5 additional media query element examples that can be added to CSS, with each item as a short bullet point:

    • max-width: 768px
      (for targeting screens up to 768 pixels wide) 
    • min-height: 600px
      (for targeting screens with a minimum height of 600 pixels) 
    • orientation: landscape
      (for targeting devices in landscape orientation) 
    • resolution: 2dppx
      (for targeting high-resolution displays with a 2x pixel density) 
    • pointer: coarse
      (for targeting devices with a coarse pointer, like a touchscreen)
